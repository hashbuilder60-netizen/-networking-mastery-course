# Course Materials

This folder contains full teaching materials beyond module outlines.

## Contents

- `NETWORKING-COURSE-HANDBOOK.md`: Chapter-by-chapter textbook-style course content.
- `learning-guides/`: Full teaching guides with examples, drills, and quiz answers.
- `wireshark-lessons/`: Packet-by-packet capture analysis lessons by module.
- `module-packs/`: Deep notes for each module.
- `practice-sets/`: Worksheets, drills, and scenario tasks.
- `answer-keys/`: Suggested answers and reasoning frameworks.
- `core-study-playbook.md`: How to study effectively across the full course.

## How to Use

1. Read the corresponding `docs` module overview.
1. Study the matching chapter in `NETWORKING-COURSE-HANDBOOK.md`.
1. Use `learning-guides` for worked examples and structured drills.
1. Run `wireshark-lessons` for packet-level understanding.
1. Complete the related lab and practice set.
1. Check your reasoning with the answer key.
