# Capstone Project: Automation and Observability

## Scenario
Build an operations toolkit that automates repetitive network tasks and provides actionable observability.

## Requirements
- Inventory-based config generation
- Nightly config backup and drift detection
- Service health checks and trend reporting
- Alert thresholds and incident playbook integration

## Deliverables
- Script set with README and usage docs
- CI workflow for lint + syntax checks
- Monitoring dashboard design notes
- Monthly operations report template
