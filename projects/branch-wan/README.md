# Capstone Project: Branch WAN

## Scenario
Design WAN connectivity for HQ and 5 branches with application prioritization, failover, and secure transport.

## Requirements
- Primary and backup WAN links per branch.
- QoS policy for voice and critical apps.
- VPN policy intent for sensitive traffic.
- Centralized monitoring and alerting.

## Deliverables
- WAN architecture decision matrix
- Routing and failover policy
- QoS class map design
- Validation tests for failover and performance
