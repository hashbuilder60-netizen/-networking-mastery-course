# Course Roadmap

## Phase 1: Core Fundamentals (Weeks 1-4)
- Module 01: Foundations
- Module 02: Models and Encapsulation
- Module 03: IPv4/IPv6 Addressing and Subnetting

## Phase 2: LAN and Routing (Weeks 5-9)
- Module 04: Switching, VLANs, STP, EtherChannel
- Module 05: Static and Dynamic Routing (OSPF focus)
- Module 06: Transport Layer and Application Behavior

## Phase 3: Services and Security (Weeks 10-13)
- Module 07: DHCP, DNS, NAT, NTP, QoS basics
- Module 08: Wireless architecture and operations
- Module 09: Security, ACLs, segmentation, hardening

## Phase 4: Modern Operations (Weeks 14-17)
- Module 10: Automation and programmability
- Module 11: SDN and cloud networking
- Module 12: Observability and troubleshooting

## Phase 5: Enterprise Readiness (Weeks 18-20)
- Module 13: WAN and VPN
- Module 14: Advanced enterprise design
- Module 15: Career prep and cert strategy

## Milestones
- M1: Build a small LAN with VLANs and inter-VLAN routing.
- M2: Deploy OSPF and troubleshoot adjacency/routing issues.
- M3: Implement security policy with ACLs and segmentation.
- M4: Automate backup and compliance checks.
- M5: Complete capstone enterprise network project.
