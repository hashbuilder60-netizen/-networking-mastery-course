# Quiz Bank Blueprint

Each module quiz should include:
- 5 concept questions
- 5 protocol behavior questions
- 5 troubleshooting scenario questions
- 3 design tradeoff questions
- 2 command/output interpretation questions

Difficulty progression:
- Modules 1-5: foundational and direct
- Modules 6-10: mixed reasoning
- Modules 11-15: architecture and operations focus
