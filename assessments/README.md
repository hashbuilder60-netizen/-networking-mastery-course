# Assessment Framework

## Types

- Quizzes: conceptual understanding and scenarios
- Practical Exams: timed config + troubleshooting
- Capstone Reviews: architecture and operational rigor

## Grading Weights

- Module quizzes: 30%
- Labs and practicals: 45%
- Capstone projects: 25%

## Minimum Passing

- Overall: 75%
- Practicals: must pass independently
- Capstones: must meet all mandatory requirements
