# Books and Learning Sources

## Recommended Books

- Practical networking fundamentals textbooks
- Enterprise routing and switching design books
- Network security operations and architecture books
- Python automation for network engineers books

## Practice Sources

- Packet Tracer challenge labs
- GNS3/EVE-NG scenario repositories
- Capture analysis exercises with Wireshark
- Timed troubleshooting drills

## Study Discipline

- Use active recall, not passive reading.
- Write short summaries after each session.
- Redo key labs without notes every 2 weeks.
