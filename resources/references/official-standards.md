# Official Standards and RFC Focus List

## Core Protocol Families

- IP addressing and forwarding behavior
- ARP and IPv6 NDP fundamentals
- TCP transport reliability and control
- OSPF operational model
- BGP path selection concepts
- DHCP and DNS behavior
- NAT terminology and behavior

## How to Study Standards

1. Read the problem statement first.
2. Identify message formats and state transitions.
3. Build a minimal lab to reproduce behavior.
4. Capture packets to verify understanding.
5. Summarize assumptions and caveats.
