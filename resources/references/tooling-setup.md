# Tooling and Environment Setup Guide

## Local Tool Stack

- Packet Tracer or GNS3/EVE-NG
- Wireshark
- Python 3 + pip
- Git
- Markdown editor

## Setup Baseline

1. Install virtualization or emulation tools.
2. Create a dedicated lab folder structure.
3. Configure Python virtual environment.
4. Clone repository and verify paths.
5. Test one baseline topology and packet capture.

## Operational Safety

- Never test destructive scripts on production devices.
- Use lab inventories separate from production inventories.
- Keep config backups before and after every change.
