# Sample Topologies and Diagram Guide

## Topology Set

- Small Office (1 router, 1 switch, 1 AP)
- Branch Office (dual WAN edge, local services)
- Campus (core, distribution, access)
- Hybrid (on-prem to cloud transit)

## Diagram Rules

- Label every interface and subnet.
- Distinguish management plane and data plane.
- Mark redundancy and failover paths.
- Show security boundaries and policy checkpoints.

## Suggested Deliverables per Project

- L2 logical diagram
- L3 routing diagram
- Security zone diagram
- Failure-domain diagram
- Operations dependency map
