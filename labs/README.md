# Lab Index

## Beginner Labs (Packet Tracer)
1. LAN basics and default gateway behavior
2. VLAN segmentation and trunking
3. Inter-VLAN routing and DHCP relay
4. Static routing fundamentals
5. OSPF single area introduction

## Intermediate/Advanced Labs (GNS3/EVE-NG)
1. Multi-area OSPF with summarization
2. ACL policy implementation and validation
3. NAT/PAT internet edge simulation
4. Site-to-site VPN concept walkthrough
5. End-to-end outage troubleshooting challenge

## Automation Labs (Containerlab + Scripts)
1. Inventory-driven config generation
2. Backup and compliance checks
3. Interface state audit and report generation

## Lab Completion Rules
- Save startup and running configs before/after changes.
- Capture verification outputs.
- Record at least one failure scenario and fix path.
